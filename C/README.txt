Für das Kompilieren der Programme benutzen sie bitte die Makefile.

Für das Testen der Programme / skripte wird arnold benutzt.

In der bash hat es folgenden aufruf: ./arnold ./ZUTESTENDESSKRIPTBZWPROGRAMM beispieltest.arnold

Sollten bei Ihnen Tests fehlschlagen benutzen sie den parameter -c damit arnold nicht stehenbleibt.
---> ./arnold -c ./ZUTESTENDESSKRIPTBZWPROGRAMM beispieltest.arnold

weiteres zur benutzung finden Sie unter: https://intern.fh-wedel.de/~mhe/arnold/
